# Practica1 - Flujo Básico de git y github
Primera Practica en un repositorio nuevo


# 1 ¿Qué es una Base de Datos Relacional?
Es una forma de guardar información de manera ordenada usando tablas que se pueden conectar entre sí. Sirve para manejar datos relacionados, como clientes y sus compras, de una forma clara y organizada.

# 2 ¿Qué es una Tabla en una Base de Datos Relacional?
Es donde se guardan los datos. Funciona como una hoja de Excel: tiene columnas y filas, y cada tabla representa algo específico, como personas, productos o pedidos.

# 3 Qué es una relación en Bases de Datos Relacionales?
Es la conexión que existe entre dos tablas. Gracias a esta relación, los datos de una tabla pueden enlazarse con los de otra, evitando repetir información innecesaria.

# 4 ¿Qué representa una columna?
Una columna es cada tipo de dato que se guarda en la tabla. Por ejemplo, nombre, correo, edad o precio. Define qué información va a tener cada registro.

# 5 ¿Qué representa un renglón?
Un renglón es un registro completo. Es decir, toda la información de una sola persona, producto o elemento dentro de la tabla.

# 6 ¿Qué es un motor de base de datos relacional?
Es el programa que se encarga de manejar la base de datos. Permite guardar los datos, buscarlos, actualizarlos y asegurarse de que todo funcione correctamente.

# 7 ¿Qué es un motor de base de datos relacional?
Es el programa que se encarga de manejar la base de datos. Permite guardar los datos, buscarlos, actualizarlos y asegurarse de que todo funcione correctamente.

# 8 ¿Cuál es el motor de base de datos ligero que no necesita un servidor?
SQLite. Es muy práctico porque funciona como un archivo y no necesita instalar un servidor, por eso se usa mucho en apps pequeñas y móviles.

# 9 Qué significa SQL en las bases de datos relacionales?
SQL es el lenguaje que se usa para comunicarse con la base de datos. Con él puedes pedir información, guardarla, modificarla o eliminarla.

# 10 ¿Cuáles son los elementos de un sistema de información basado en bases de datos relacionales?
Incluye la base de datos, el motor que la maneja, el lenguaje SQL, la aplicación que usa los datos, las personas que interactúan con el sistema y el equipo donde todo funciona.
